# 쏙쏙 들어오는 함수형 코딩 - 에릭 노먼드

## 읽는 이유

코드를 짜며 다른 사람이 보더라도 괜찮을 코드일까 하는 생각들을 많이 한다. 읽어줄 누군가가 없어 혼자 판단을 해야하는데 내가 하는 판단이 맞는지 모를 때가 많다.

이 책을 통해 내 판단이 정확에 가까워지기를 바라는 마음으로 읽어보려고 함.
